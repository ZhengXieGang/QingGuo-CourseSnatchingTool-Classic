# 适用于老版青果教务系统的选课抢课工具
## 青果选课平台 Beta


<img width="100%" alt="pic" src="https://github.com/user-attachments/assets/6d931afc-0e83-436d-8fe4-48f589abae96" />

基于 Flask 的老版青果教务系统选课辅助工具，单文件部署，支持自动抢课、风控等待、Session 保活。
适合校内教务系统课程监控和自动提交，或学校服务器被挤爆前端加载不出来的场景。

## 功能

- 登录 / 挂断 / 失效自动重登
- 课程与班级列表拉取、动态筛选
- 自动抢课循环（监控人数 → 有余量立即提交 → 可选验证）
- 风控识别与自动等待倒计时
- Session 保活（15 分钟自动续期）
- 登录延时执行 / 失败自动重试
- 请求耗时、网络延迟、请求频率实时展示
- 预设账号自动填充
- 日志面板实时查看运行状态

## 快速开始

### 自动化配置工具

默认配置不适用，需使用自带的脚本一键获取配置：

```bash
# 用法
python kingo_prober.py <学校教务系统登录页URL>

# 示例
python kingo_prober.py https://jw.example.edu.cn/jwweb/MAINFRM.aspx
```

该工具会通过自动生成配置信息。你只需运行探测脚本，将输出的代码块完整替换到 `app.py` 顶部的“常量配置”对应区域即可。

注意：对于新版青果（CAS 2018年版及其他版本）未进行适配，因为我学校还在用清朝老版本，想试试自己适配可参考`CAS_ADAPTATION_GUIDE.md`。

**环境要求：** Python 3.10+

```bash
# 安装依赖
pip install -r requirements.txt
# 或 Windows 运行 install_deps.cmd

# 启动
python app.py
# 浏览器自动打开 http://127.0.0.1:5000
```

## 使用流程

1. 登录教务系统
2. 拉取课程列表，展开查看班级
3. 选择目标班级，设置抢课间隔
4. 启动抢课，日志面板查看状态


## 注意事项

- 控制请求频率，避免触发风控
- 不同版本青果教务页面结构可能不同，需适配
- 仅供个人学习和合理范围内使用
